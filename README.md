# 🧠 SkimLit – Medical Abstract Classification

🚀 A deep learning NLP project to classify medical research abstract sentences into structured sections.

## 📌 Overview
This project builds and compares machine learning and deep learning models to classify sentences from medical abstracts into predefined categories.


## 📊 Dataset
- 📚 PubMed 200K RCT dataset  
- 🔢 ~200,000 labeled sentences  
- 🏷️ 5 Classes:
  - Background
  - Objective
  - Methods
  - Results
  - Conclusions  


## 🛠️ Models Implemented
- 📌 TF-IDF + Logistic Regression (Baseline)
- 🧠 1D CNN
- 🔁 LSTM
- 🔀 Hybrid Model (Token + Character Embeddings)


## ⚙️ Features
- 🧹 Text preprocessing (tokenization, padding)
- 🔡 Character-level + word-level embeddings
- 🧩 Multi-input deep learning architecture
- 📈 Model comparison and evaluation


## 💻 Tech Stack
- 🐍 Python  
- 🤖 TensorFlow  
- 📊 Scikit-learn  
- 🔢 NumPy, Pandas  


## 📈 Results
- Deep learning models outperform baseline approaches  
- Hybrid embeddings improve contextual understanding
